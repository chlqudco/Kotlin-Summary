- 인라인 클래스
	- 래퍼클래스의 문제점은 런타임 부가 비용이 든다
		- 새로운 클래스가 생성될 때 마다 래퍼 클래스를 생성해야 함
	- 이 문제를 해결하기 위해 인라인 클래스를 도입함
		- 부가 비용 없이 쓸 수 있기 때문에 value 클래스라고 부름

- 인라인 클래스 정의하기
	- 이름 앞에 value class 라고 정의하면 끝
	- 주 생성자에는 불변 프로퍼티를 하나만 선언해야 함

- 부호 없는 정수
	- 갑자기 뜬금없이 이게 왜 나오지?
	- 부호 없는 정수는 타입 앞에 U를 붙이면 됨.
		- UByte, ULong 등등
	- 부호 있는 타입에 대입 불가능
